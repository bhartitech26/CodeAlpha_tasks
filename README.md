# Project 2: Hangman Game 🎮

## Description

The Hangman Game is a text-based word guessing game built using Python. The player must guess a randomly selected word one letter at a time before running out of attempts.

## Features

- Random word selection.
- Tracks correct and incorrect guesses.
- Maximum 6 incorrect attempts.
- Displays word progress.
- Prevents duplicate letter entries.
- Interactive console gameplay.

## Technologies Used

- Python 3
- Random Module

## Concepts Covered

- Variables
- Lists
- Strings
- Loops
- Conditional Statements
- Random Module

## Game Rules

1. The computer selects a random word.
2. The player guesses one letter at a time.
3. Correct letters are revealed in the word.
4. Incorrect guesses reduce remaining attempts.
5. The player gets 6 wrong attempts.
6. The game ends when the word is guessed or attempts run out.

## Sample Output

```text
=== Welcome to Hangman ===

Word: _ _ _ _ _ _

Enter a letter: p
Correct guess!

Word: p _ _ _ _ _
```

# Project 3: Stock Portfolio Tracker 📈

## Description

The Stock Portfolio Tracker is a Python application that calculates the total value of a user's stock investments using predefined stock prices. It also provides an option to save portfolio details to a text file.

## Features

- Accepts stock symbols and quantities.
- Uses dictionary-based stock price storage.
- Calculates individual stock values.
- Calculates total investment value.
- Handles invalid stock symbols.
- Saves portfolio reports to a text file.

## Technologies Used

- Python 3
- Dictionaries
- Loops
- Conditional Statements
- File Handling

## Concepts Covered

- Dictionary Data Structure
- User Input and Output
- Iteration
- Arithmetic Calculations
- Conditional Logic
- File Handling

## Example Output

```text
----- Portfolio Summary -----

AAPL: 10 shares × $180 = $1800
TSLA: 5 shares × $250 = $1250

Total Investment Value = $3050
```
